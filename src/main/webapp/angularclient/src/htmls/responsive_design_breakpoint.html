Responsive Design Breakpoints

How to distinguish between mobile, tablet, and desktop

Understanding Responsive Breakpoints

Responsive design uses CSS media queries to adapt layout to different screen sizes. The mobile-first approach means we start with styles for mobile devices, then add styles for larger screens.

/* Mobile First Approach - Base styles */
.container {
  padding: 10px;
}

/* Tablet - 768px and up */
@media (min-width: 768px) {
  .container {
    padding: 20px;
    display: grid;
    grid-template-columns: 1fr 1fr;
  }
}

/* Desktop - 1024px and up */
@media (min-width: 1024px) {
  .container {
    grid-template-columns: 1fr 1fr 1fr;
    max-width: 1200px;
    margin: 0 auto;
  }
}
