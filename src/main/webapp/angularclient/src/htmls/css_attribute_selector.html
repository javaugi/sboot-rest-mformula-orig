CSS Attribute Selector

Understanding a[href|="en"]

How the Selector Works

The CSS selector a[href|="en"] targets all <a> elements whose href attribute value:

  1. Is exactly "en"
  2. OR begins with "en" immediately followed by a hyphen (-)
/* This selector styles links where href starts with "en" followed by hyphen */
a[href|="en"] {
    background-color: #ffeaa7;
    color: #d35400;
    font-weight: bold;
    border: 2px solid #fdcb6e;
    border-radius: 4px;
}

Demonstration

Below you can see examples of links that will and won't be styled by the a[href|="en"] selector:

These Links WILL Be Styled

  • href="en"
  • href="en-US"
  • href="en-GB"
  • href="en-au"
  • href="en-001"

These Links WILL NOT Be Styled

  • href="enUS" (no hyphen)
  • href="english" (no hyphen)
  • href="en.html" (no hyphen)
  • href="fr" (different prefix)
  • href="en_US" (underscore instead of hyphen)

Note: The selector is particularly useful for styling language-specific links based on the language code pattern (like en-US, en-GB, etc.).

Practical Applications

The [attr|="value"] selector is especially useful for:

  • Styling language-specific links based on language codes (e.g., en, en-US, es, es-MX)
  • Targeting specific protocol versions (e.g., http-1, http-2)
  • Selecting elements with specific prefix patterns followed by a hyphen
