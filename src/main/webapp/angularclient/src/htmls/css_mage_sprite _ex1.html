CSS Image Sprite Example

Multiple images in one file with CSS controls

An image sprite is a collection of images put into a single image file. Using CSS, we can control which part of the image is displayed. This technique reduces the number of server requests, making websites load faster.

How it works - CSS Code:

/* The sprite image contains all icons */
.sprite {
    background-image: url('sprite.png');
    background-repeat: no-repeat;
    height: 64px;
    width: 64px;
}

/* Each icon is positioned using background-position */
.facebook {
    background-position: 0 0;
}

.twitter {
    background-position: -64px 0;
}

.instagram {
    background-position: -128px 0;
}
