Visual explanation with interactive demo

Box Shadow Syntax

The CSS box-shadow property applies shadow effects to an element.

Syntax:

box-shadow: h-offset v-offset blur-radius spread-radius color inset;

Parameters:

  • h-offset: Horizontal shadow position (required)
  • v-offset: Vertical shadow position (required)
  • blur-radius: Blur amount (optional, default 0)
  • spread-radius: Shadow size (optional, default 0)
  • color: Shadow color (optional, default black)
  • inset: Changes shadow to inner shadow (optional)
